# Animation: Project Showcase Cards

A portfolio-style project showcase component featuring image sections, hover elevation, and action button reveals.

## Features

- Pure HTML & CSS
- Project gallery layout
- Hover elevation effect
- Action button reveal
- Responsive grid
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="project-card">

    <div class="project-image">
        🚀
    </div>

    <div class="project-content">
        <h3>Project Name</h3>
    </div>

</div>
```

## Why is it useful?

Project showcase cards are widely used in portfolios, startup websites, product galleries, and documentation pages. This example provides a clean way to highlight projects while adding subtle motion and interactivity.
